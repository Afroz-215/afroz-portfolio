UI/UX Design

I am a fresher passionate about designing user interfaces that are both attractive and easy to use. I focus on applying user-centered design principles to create seamless, intuitive layouts that improve accessibility and user satisfaction.

My approach includes:

  • Creating wireframes and prototypes to visualize user flows.
  • Designing clean and consistent visual elements.
  • Ensuring usability and accessibility across devices.
  • Collaborating closely with developers to implement designs effectively.

I continuously work on improving my skills by studying design trends and tools like Figma and Adobe XD.
