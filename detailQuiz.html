SmartQuiz

SmartQuiz is a dynamic web-based quiz application built using HTML, CSS, and JavaScript. It features a clean and responsive interface optimized for both desktop and mobile users.

Key features include:

  • Timed questions to challenge users and increase engagement.
  • Instant score calculation and result display.
  • Answer validation with immediate feedback.
  • Modern, mobile-friendly UI design for a smooth experience.

This project showcases my skills in frontend web development and creating interactive, user-focused applications.
