Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built using HTML, CSS, and JavaScript, offering a fun and interactive experience with a polished UI.

Key Features:

  • 3x3 grid with real-time gameplay
  • Turn-based logic and UI feedback
  • Winner and draw detection
  • Disabled cells after play
  • Reset & new game functionality

This project showcases DOM manipulation, condition handling, and a clean interactive design.
