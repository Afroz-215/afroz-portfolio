Tic-Tac-Toe Game

A classic Tic-Tac-Toe game implemented using HTML, CSS, and JavaScript, designed for an engaging and responsive experience.

Key features:

  • 3x3 grid layout with real-time interaction
  • Turn-based logic and clear UI feedback
  • Automatic winner detection and draw handling
  • Disabled cells after play to prevent re-clicking
  • Reset and new game functionality for replay

This project highlights interactive game logic, clean UI design, and efficient JavaScript event handling.
